La clave no es el JSON

La tendencia es pedirle a la IA el JSON de una imagen. Eso ayuda, pero no basta. El JSON contiene la estructura visual que la IA detecta. Tu marca aparece cuando esa estructura se adapta a tus anclas.

Idea central

Copiar una foto no crea una marca. Lo que crea marca es extraer la esencia visual de una referencia y transformarla con tus elementos fijos: rostro, colores, fondo, estilo, composición y símbolos propios.

No es copiar la foto.
Es extraer su esencia
y adaptarla a tu marca.
Regla ESG: una referencia inspira. Tus anclas dirigen. El resultado debe parecer parte de tu universo visual, no una copia de la imagen original.
Referencia

Una imagen que te inspira por composición, atmósfera, pose, iluminación o estilo.

JSON

La estructura visual detectada: encuadre, luz, paleta, objetos, posición, fondo y energía.

Ancla

Los elementos que hacen que el resultado sea tuyo: identidad, color, fondo, marca y símbolos.

El proceso ESG
de una foto a tu marca

Un flujo de cuatro pasos para convertir cualquier referencia visual en una imagen coherente, profesional y alineada a tu identidad.

1

Referencia

Usas una imagen de referencia que te inspira. No la copias. La analizas como punto de partida visual.

2

Extraes el JSON

Le pides a la IA que detecte la estructura visual completa: composición, ángulo, luz, paleta, fondo, objetos, pose y atmósfera.

3

Aplicas tus anclas

Adaptas el JSON a tu rostro, colores, fondo fijo, estilo de marca y elementos propietarios.

4

Resultado ESG

Generas imágenes coherentes, alineadas a tu marca, listas para posicionar, vender o construir presencia visual.

Mis anclas fijas

Completa estos campos para construir tu prompt maestro. Mientras más precisas sean tus anclas, menos genérico será el resultado.

Prompts listos

Usa estos prompts en Claude, ChatGPT o cualquier IA que permita analizar imágenes. Primero extrae, después adapta, luego genera.

Prompt 1 — Extraer JSON visual
Voy a subir una imagen de referencia.

Extrae el JSON visual completo de esta imagen.

No quiero que describas la imagen de forma superficial. Quiero una estructura visual útil para recrear la esencia de la referencia sin copiarla.

Analiza y entrega el JSON con estas secciones:

1. COMPOSICIÓN
- encuadre
- orientación
- posición del sujeto
- profundidad
- distribución de elementos

2. ILUMINACIÓN
- tipo de luz
- dirección
- intensidad
- sombras
- reflejos

3. PALETA VISUAL
- colores dominantes
- colores secundarios
- contraste
- temperatura visual

4. SUJETO / AVATAR / PRODUCTO
- postura
- expresión
- proporciones visibles
- ropa
- detalles importantes

5. FONDO Y AMBIENTE
- locación
- textura
- atmósfera
- elementos de apoyo

6. ESTILO VISUAL
- estética
- nivel de realismo
- sensación emocional
- tipo de marca que comunica

7. ELEMENTOS QUE NO DEBEN COPIARSE
Identifica qué partes pertenecen a la imagen original y no deben repetirse literalmente.

8. ESTRUCTURA REUTILIZABLE
Convierte todo en un JSON limpio que pueda ser adaptado a otra identidad visual.
Prompt 2 — Adaptar JSON a mis anclas
Toma el JSON visual extraído de la referencia y adáptalo a mis anclas de identidad.

MIS ANCLAS FIJAS:
Avatar base: [usaré una imagen base / descripción del avatar]
Colores de ropa: [escribir aquí]
Fondo fijo: [escribir aquí]
Estilo de marca: [escribir aquí]
Elementos de marca: [escribir aquí]
Uso final de la imagen: [escribir aquí]

REGLA PRINCIPAL:
Mantén estas anclas en todos los JSON que extraigas de futuras referencias.
No quiero copiar la imagen original.
Quiero conservar únicamente la estructura visual de la referencia y transformarla para que parezca parte de mi propia marca.

ENTRÉGAME:
1. JSON original detectado resumido.
2. JSON adaptado a mis anclas.
3. Lista de cambios realizados para mantener coherencia visual.
4. Prompt final listo para generar la nueva imagen.
Prompt 3 — Generar imagen final
Voy a subir mi imagen base y el JSON adaptado.

Genera una nueva imagen utilizando la estructura visual del JSON adaptado, pero manteniendo mis anclas de identidad.

REGLAS:
- No copies la imagen de referencia original.
- Usa solo su estructura visual: composición, luz, atmósfera, pose y dirección estética.
- Mantén mi avatar / rostro / personaje / producto según la imagen base.
- Mantén mis colores de ropa, fondo fijo, estilo de marca y elementos propietarios.
- El resultado debe sentirse como parte de mi marca, no como una imagen aislada.
- Evita fondos genéricos, cambios de identidad, deformaciones, textos mal escritos o elementos que no pertenezcan a mi sistema visual.

ENTREGA:
Una imagen final coherente con mi identidad visual y lista para usar como pieza de marca.

Uso final

Cómo aplicar el protocolo

1

Abre un nuevo chat en modo imagen

Empieza limpio para que la IA no arrastre instrucciones viejas.

2

Sube tu foto base

Puede ser tu rostro, avatar, producto o elemento fijo de marca.

3

Pega el JSON adaptado

No pegues solo la referencia. Pega la referencia transformada por tus anclas.

4

Genera tu nueva imagen

Evalúa si se mantiene la misma esencia, misma marca y mismo sistema visual.

5

Repite con nuevas referencias

Usa referencias distintas, pero conserva siempre las mismas anclas.
