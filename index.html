El fin de los prompts que salen mal

La clave no es el JSON

La tendencia en redes es pedirle a ChatGPT el JSON de una imagen. Pero copiar una estructura no crea una marca. Lo que crea marca es adaptar esa estructura a tus anclas de identidad.

Referencia

La imagen te inspira y muestra una composición, luz, pose, atmósfera o dirección que quieres usar como punto de partida.

JSON

El JSON contiene la estructura visual que la IA está viendo. Es una lectura técnica, no una identidad de marca.

Ancla

El ancla fija lo que no debe cambiar: avatar, colores, fondo, estilo, elementos de marca y esencia visual.

No es copiar la foto.
Es extraer su esencia y adaptarla a tu marca.

Regla principal

          
Misma esencia. Misma marca. Infinitas referencias.

          
Mantén estas anclas en todos los JSON que extraigas de futuras referencias. No quieres copiar la imagen original. Quieres conservar únicamente la estructura visual de la referencia y transformarla para que parezca parte de tu propia marca.

        

        

          
Uso final

          
Cómo usarlo

          

            
1. Abre un nuevo chat en modo imagen.

            
2. Sube tu foto base.

            
3. Pega el JSON adaptado.

            
4. Genera tu nueva imagen.

            
5. Repite el proceso con nuevas referencias manteniendo las mismas anclas.
