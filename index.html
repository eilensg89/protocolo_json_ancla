Protocolo JSON Ancla

Este es el flujo correcto para generar imágenes con IA sin perder identidad: primero se extrae el escenario como JSON, luego se bloquea el sujeto con referencias A y B, después se aplican solo los cambios autorizados y al final se genera con las tres imágenes en orden.

SistemaESG = Escenario, Sujeto, GuíaFlujoQué subir y cuándo subirloReferenciasImagen A, B y C sin confusiónPromptsJSON listos para copiar

Qué significa ESG aquí

Este protocolo no trabaja con un prompt suelto. Trabaja con JSON estructurado porque la IA necesita jerarquía, bloqueos y reglas de prioridad.

Escenario

Viene de la referencia Pinterest. Controla pose, encuadre, composición, cámara, fondo, luz, objetos, atmósfera y estilo visual.

Sujeto

Viene de las fotos del usuario. Imagen A bloquea rostro e identidad. Imagen B bloquea cuerpo, proporciones y presencia física.

Guía

Controla cambios autorizados: ropa, color, fondo, ambiente, calidad y restricciones. Si no está autorizado, no se cambia.

Regla madre: Imagen C no puede cambiar rostro ni cuerpo. Imagen C solo guía escena. A y B mandan sobre el sujeto.

Flujo operativo paso a paso

Sigue este orden. Si se mezclan las referencias desde el inicio, la IA confunde sujeto con escena.

1

Sube solo la referencia Pinterest

Subes Imagen C. Pides extraer el JSON exacto de la imagen. Este JSON se llama JSON 1. Aquí no se cambia rostro, cuerpo, ropa ni escenario. Solo se extrae la estructura visual.

2

Sube rostro + cuerpo base

Subes Imagen A + Imagen B y pegas el JSON 1. Pides que se sustituya el sujeto original del JSON 1 por la persona de las imágenes A y B. El resultado se llama JSON 2.

3

Ajusta solo lo pedido

Subes o pegas el JSON 2. Pides cambios específicos: color de ropa, estilo de ropa, fondo o ambiente. Si no pides cambiar escenario, el escenario queda intacto. El resultado es el JSON final.

4

Genera la imagen final

Subes las imágenes en este orden: A rostro, B cuerpo, C escena. Pegas el JSON final. La IA genera usando A para identidad, B para cuerpo y C para escena, pose, luz y composición.

Orden exacto de referencias

Esta sección debe aparecer antes de los prompts para que el usuario sepa exactamente qué imagen subir en cada paso.

Imagen A
Rostro

Foto del rostro del usuario. Activa IDENTITY_LOCK: rasgos, cabello, piel, expresión base e identidad reconocible.

Imagen B
Cuerpo

Foto del cuerpo base. Activa BODY_LOCK: proporciones, complexión, altura percibida, postura base y presencia física.

Imagen C
Escena

Referencia Pinterest. Activa SCENE_LOCK: pose, encuadre, fondo, objetos, luz, cámara, composición y ambiente.

No inviertas el orden: si la IA usa Imagen C como referencia de sujeto, copiará la persona de Pinterest. C solo guía la escena.

Bloqueos que la IA sí entiende

Estas llaves convierten el proceso en una instrucción jerárquica, no en una solicitud vaga de “parecido”.

IDENTITY_LOCK

Rostro, cejas, ojos, nariz, labios, tono de piel, cabello, edad percibida, expresión e identidad reconocible.

BODY_LOCK

Proporciones corporales, complexión, altura percibida, volumen, postura base y presencia física.

SCENE_LOCK

Pose, cámara, composición, iluminación, fondo, ambiente, objetos y distribución de elementos.

QUALITY_LOCK

8K, máxima nitidez al hacer zoom, cero ruido, cero grano, piel natural, textura real y microdetalle limpio.

Completa los campos

Al completar estos campos, los prompts de JSON 2, ajustes y generación final se actualizan automáticamente.

Voy a subir una sola imagen de referencia.

Esta imagen será la IMAGEN C = referencia de escena / Pinterest.

TAREA:
Extrae el JSON visual exacto de la Imagen C.

En esta etapa NO uses rostro ni cuerpo del usuario.
En esta etapa NO hagas cambios creativos.

Validación final
Checklist antes de aprobar una imagen
Sujeto
¿El rostro viene de Imagen A? ¿El cuerpo viene de Imagen B? ¿La referencia C no contaminó identidad, edad percibida ni proporciones?
Escena
¿La pose, composición, fondo, luz y objetos vienen de Imagen C? ¿Se mantuvo el escenario si no se pidió cambiarlo?
Guía
¿Solo cambió lo autorizado? ¿El color y estilo de ropa respetan la guía? ¿No se inventaron objetos nuevos?
Calidad
¿La imagen tiene nitidez máxima al hacer zoom? ¿No hay ruido, grano, piel plástica ni textura borrosa?
