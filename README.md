## Corrección aplicada
Esta versión deja claro el flujo completo:
1. Subir Imagen C / Pinterest para extraer JSON 1.
2. Subir Imagen A / rostro + Imagen B / cuerpo base y pegar JSON 1 para crear JSON 2.
3. Pegar JSON 2 y aplicar solo cambios autorizados.
4. Subir Imagen A, B y C en orden + pegar JSON final para generar la imagen.

## Sistema ESG aplicado
- Escenario = SCENE_LOCK = Imagen C.
- Sujeto = IDENTITY_LOCK + BODY_LOCK = Imagen A + Imagen B.
- Guía = CHANGE_REQUEST + QUALITY_LOCK = instrucciones del usuario + calidad.

Incluye instrucciones visibles en la landing y dentro de cada prompt.
